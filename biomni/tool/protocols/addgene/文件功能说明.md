这个文件夹下**不是代码文件**，而是一组 **Addgene 实验 protocol 的纯文本整理文件**。目录里全部都是 `.txt`，文件名直接对应具体实验主题；我抽查了 `AAV Production`、`Gibson Assembly Protocol`、`Lentivirus Production Protocol`，内容格式都是“PROTOCOL / SOURCE / SUMMARY 或 INTRODUCTION / EQUIPMENT / REAGENTS / PROCEDURE”这种协议文档结构，不是 Python 实现。([GitHub][1])

可以把这个目录理解成 Biomni 的 **本地实验操作知识库**：把 Addgene 网站上的常用分子生物学、病毒包装、细胞实验、基础实验室操作 protocol，预先保存为文本，供上层工具或 agent 检索、读取和回答实验步骤问题。目录中文件名覆盖了 AAV、lentivirus、plasmid cloning、transfection、PCR、凝胶、电泳、细菌培养、安全规范等主题。([GitHub][1])

下面按文件逐个说明其功能。我会分成几类，这样更容易看。

## 1）AAV / 病毒载体相关

* `Addgene_ AAV Production.txt`：AAV 生产 protocol，主要讲 HEK293T 细胞扩增、转染、收获病毒颗粒的流程。([GitHub][1])
* `Addgene_ AAV Purification.txt`：AAV 纯化 protocol，作用是把生产后的 AAV 样品进一步纯化。文件名已明确其用途。([GitHub][1])
* `Addgene_ AAV Titration by qPCR.txt`：用 qPCR 测 AAV 滴度。([GitHub][1])
* `Addgene_ AAV ddPCR Titration.txt`：用 ddPCR 测 AAV 滴度。([GitHub][1])
* `Addgene_ Lentivirus Production Protocol.txt`：lentivirus 生产 protocol，和 AAV production 类似，偏慢病毒包装与收获。([GitHub][1])
* `Addgene_ Lentivirus ddPCR Titration.txt`：用 ddPCR 测 lentivirus 滴度。([GitHub][1])
* `Addgene_ Colony Formation Titering Assay.txt`：用集落形成实验做病毒滴度/感染效力相关测定。([GitHub][1])
* `Addgene_ Fluorescence Titering Assay.txt`：用荧光法进行滴度测定。([GitHub][1])

## 2）抗体与免疫检测相关

* `Addgene_ Affinity Purification of Recombinant Antibodies with Protein A or Protein G.txt`：用 Protein A / Protein G 做重组抗体亲和纯化。([GitHub][1])
* `Addgene_ Antibody Validation Using the Indirect ELISA Method.txt`：用间接 ELISA 做抗体验证。([GitHub][1])
* `Addgene_ Coomassie Purity Stain of Recombinant Antibodies.txt`：用考马斯亮蓝染色评估重组抗体纯度。([GitHub][1])
* `Addgene_ Immunocytochemistry.txt`：免疫细胞化学 protocol，用于细胞层面的染色检测。([GitHub][1])

## 3）CRISPR / 文库相关

* `Addgene_ CRISPR Library Amplification.txt`：CRISPR 文库扩增 protocol，通常用于 pooled library 的扩增和维持复杂度。([GitHub][1])
* `Addgene_ Isolating a Monoclonal Cell Population by Limiting Dilution.txt`：极限稀释法分离单克隆细胞群，常用于基因编辑后克隆筛选。([GitHub][1])

## 4）质粒克隆 / DNA 操作核心流程

这部分是这个目录里最典型的一大组，适合做分子克隆基础流程知识库。([GitHub][1])

* `Addgene_ Gibson Assembly Protocol.txt`：Gibson assembly 拼接 protocol。([GitHub][1])
* `Addgene_ Ligation Independent Cloning.txt`：LIC，无连接酶克隆。([GitHub][1])
* `Addgene_ Molecular Biology Protocol - Restriction Digest of Plasmid DNA.txt`：质粒 DNA 限制性内切酶消化。([GitHub][1])
* `Addgene_ Plasmid Cloning by PCR (with Protocols).txt`：基于 PCR 的质粒克隆流程。([GitHub][1])
* `Addgene_ Plasmid Cloning by Restriction Enzyme Digest (with Protocols).txt`：基于酶切的经典质粒克隆。([GitHub][1])
* `Addgene_ Plasmid modification by Annealed Oligo Cloning by.txt`：用退火寡核苷酸进行质粒改造/插入。([GitHub][1])
* `Addgene_ Protocol - How to Ligate Plasmid DNA.txt`：DNA 连接反应。([GitHub][1])
* `Addgene_ Protocol - How to Perform a Diagnostic Digest.txt`：诊断性酶切，用于验证构建是否正确。([GitHub][1])
* `Addgene_ Protocol - How to Purify DNA from an Agarose Gel.txt`：从琼脂糖凝胶中回收 DNA。([GitHub][1])
* `Addgene_ Protocol - How to Run an Agarose Gel.txt`：跑胶/凝胶电泳基础 protocol。([GitHub][1])
* `Addgene_ Protocol - How to Perform Sequence Analysis.txt`：序列分析/测序结果检查流程。([GitHub][1])
* `Addgene_ Protocol - How to Design Primers.txt`：引物设计基础。([GitHub][1])
* `Addgene_ DNA Quantification.txt`：DNA 定量方法。([GitHub][1])
* `Addgene_ Handling Plasmids from Addgene - Purifying Plasmid DNA.txt`：收到 Addgene 质粒后的质粒 DNA 纯化处理指南。([GitHub][1])

## 5）细菌培养与菌株保存

* `Addgene_ Protocol - Bacterial Transformation.txt`：细菌转化 protocol。([GitHub][1])
* `Addgene_ Protocol - How to Inoculate a Bacterial Culture.txt`：接种细菌培养。([GitHub][1])
* `Addgene_ Protocol - How to Streak a Plate.txt`：平板划线分离。([GitHub][1])
* `Addgene_ Protocol - How to Create a Bacterial Glycerol Stock.txt`：甘油菌保藏。([GitHub][1])
* `Addgene_ Pouring LB Agar Plates.txt`：配制并倾倒 LB 琼脂平板。([GitHub][1])

## 6）转染 / RNA / 一般细胞实验

* `Addgene_ General Transfection.txt`：通用转染 protocol。([GitHub][1])
* `Addgene_ Kit Free RNA Extraction.txt`：无试剂盒 RNA 提取。([GitHub][1])

## 7）基础实验操作技能

这几份更像“实验室新手训练材料”或 SOP。([GitHub][1])

* `Addgene_ Centrifugation.txt`：离心基础操作。([GitHub][1])
* `Addgene_ Pipetting Protocol.txt`：移液规范。([GitHub][1])

## 8）实验室安全与规范

* `Addgene_ Lab Safety for Biosafety Levels One and Two.txt`：BSL-1 / BSL-2 实验室安全要求。([GitHub][1])
* `Addgene_ Personal Protective Equipment (PPE) for BSL-1 and BSL-2 Labs.txt`：BSL-1 / BSL-2 环境下个人防护装备要求。([GitHub][1])

### 这组文件在 Biomni 里大概起什么作用

它们更像是 **protocol 语料库**，不是执行分析的代码。上层系统可以把这些文本当作本地可检索知识源，在用户问“怎么做 AAV 生产”“Gibson Assembly 步骤是什么”“如何做细菌转化”时，直接从这些 `.txt` 协议里抽取答案。你前面看的 `biomni/tool/protocols.py` 就是协议检索层；这个 `addgene/` 子目录则像它的一个本地静态数据源。这个判断和目录位置、文件格式都一致。([GitHub][1])

### 一句话总结

这个文件夹的功能不是“放代码”，而是把 Addgene 的常用实验 protocol 按主题保存成文本文件，覆盖了：
**病毒包装与滴度、抗体纯化与验证、CRISPR 文库、分子克隆、细菌培养、细胞实验、基础操作和实验室安全**。([GitHub][1])
